An Haiku Feed Aggregator

Jan 14, 2024 by Fabio Tomat aka TmTFx

This piece of code is a simple Feed Aggregator for Haiku written in python (version 3.10).
I hope you enjoy it.
The program stores on disk (~/config/non-packaged/data/BGator2) the feed of your favorite newspapers as an attribute of a folder,
inside the newspaper folder FeedGator will create a file per news with its relative summary. You can set a timer that will retrieve all
newspapers updates.
