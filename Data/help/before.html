Before running...

FeedGator uses python3 version 3.10 and the program needs some modules:

  • Haiku-PyAPI: Binding modules for Python3 to Haiku APIs. Haiku-PyAPI is available on HaikuDepot or GitHub

  • feedparser: universal feed parser for python, used to manage feeds
