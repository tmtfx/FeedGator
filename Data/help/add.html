How to add new feeds


to add a new feed and therefore download the news of the relative newspaper you can choose either one of these 2 methods:
  1. Click on menu File and select Add Paper
  2. Click on "+" button
both methods open the same dialog. It looks like this:



You can call this window by using the Command+d shortcut.

Now you can write or paste the feed in the text field and press the "Add Feed" button to do it, or press Cancel to return to hide this dialog.
