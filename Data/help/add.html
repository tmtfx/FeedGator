How to add new feeds


to add a new feed and then download the news of the relative newspaper you can choose one of these 2 methods:
  1. Click on menu File and select Add FeedRss
  2. Click on "+" button
both methods open the same interface and the program will become like this:



Now you can write or paste the feed in the text field and press the Add button to do it
or press Cancel to return to the regular interface
