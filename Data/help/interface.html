The interface

the program in its "beauty":







a brief description of available commands:
  • in the "menu" you can find 4 courtains: File, News, Sort By and Help; in "File" you'll find commands for adding a new newspaper, remove a newspaper, change settings and quit the application; in "News" you can download all newspaper's new news, mark as read all the news of a specific newspaper and remove all the news of a specific newspaper without removing the newspaper itself; in "Sort By" you can sort the news by Name, by Unread and by Date; in "Help" you can launch this help or read something about me and the license of this program
  • the + and - buttons have the same function of adding a new newspaper and remove a newspaper found in the File menu
  • once you added at least 1 newspaper the Download button (arrow down) allows you to download the news like the command in the News menu
  • selecting a newspaper in the newspaper list (left), its news will be listed in the news list on the right. If a newspaper has unread news, the character used to write its name will be bold and the selection will appear red; if all its news are marked as read, its selection will be colored with yellow and the normal characted will be used. If you double click a newspaper its folder will be opened. If you double-click a newspaper while you keep Shift pressed a details dialog will be opened. 
  • with the buttons Mark as Read and Mark as Unread you set the "Unread" attribute of the selected news, once a news is marked as read it is not listed anymore with bold characters, a new news downloaded is always marked as Unread until you select it, but you can remark it as Unread; you can "mark as read" all the news in a newspaper by clicking News -> All as read after selecting the relative newspaper.
  • by selecting a news in the news list a summary of the news (if available) is shown in the text area below, if you want to read the entire article in your web browser you have to doubleclick the news in the news list or press the "Open in browser" button.
  • above the news list there's a progress bar that gives you some information about the status of the download process.

    • Tips & Tricks

  • Double-clicking an item in the newspaper list will open its corresponding on-disk folder where the news articles are stored.
  • By pressing SHIFT and double-clicking an item in the newspaper list will open a dialog which displays: the newspaper name, its path on disk, the feed address, and the number of contained articles (files).
  • File->Settings will open a cryptic dialog that handles some settings stored on a config file, in the "General" tab you'll see two options listed: "sort" and "minimized". While "sort" value is easily changed selecting the sorting method in "Sort By" menu, "minimized" is a hidden configuration in order for FeedGator to be launched minimized.
    The timer tab is more interesting because it allows you to configure a personalized update-interval for FeedGator to retrieve the news. With the "enable" item you can set or disable it, and with the "timer" item you can set the update interval value in microseconds, the default timer is set to 5 minutes.
